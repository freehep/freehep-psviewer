Introduction

The PS Viewer is a Level 2 PostScript interpreter which can be used as a stand-alone (Encapsulated) PostScript viewer, or it can be embedded in a user application, see usage.

Some limitations on the PostScript interpretation exist.

Examples

The following pages show PostScript examples displayed in multiple PS Viewer Applets:

  • Cookbook: examples from the Blue Book, see documentation below
  • Tutorial: examples from the Green Book, see documentation below
  • Fonts: examples with fonts
  • FreeHEP: examples of output from the FreeHEP VectorGraphics package
  • WIRED: examples of output from the WIRED Event Display package
  • Plots: examples of scientific plots, among others from the Root package
  • Resizable Plot: example of a scientific resizable plot
